        Программа представляет собой систему учёта и распределения 
        ресурсов и оценкой рисков на частном производстве растений.
        
        Пользователю предлагается два варианта: первый – подключить 
        свой json файл с уже готовой моделью набора ресурсов (или же 
        создать её в консоли), второй – ознакомиться с актуальными 
        программами по выращиванию растений и выбрать себе 
        что-то подходящее.
        
        Пользователь вводит показатели своих ресурсов, программа по 
        формулам рассчитывает выручку, риски и трудозатраты. Иначе, 
        можно ничего не вводить и просто знакомиться с разными 
        способами получения той или иной суммы денег.
        
        На данный момент планируется: чтение и запись в файлы (с возможностью 
        создания там новых объектов) , использование наследования, паттернов 
        проектирования, использование динамических структур данных, реализация 
        меню/UI с учётом возможностей.


	The program represents itself a system of accounting and distribution of 
	resources as well as estimation of risks for a planting production.
	
	For user there are two ways of interaction: first – load their JSON file with
	specified resources amount (or either type it down in console),
	second – take a look at some prepared programmes of growing plants
	to choose something fit.

	User specifies the amount of resources at their disposal and then the program
	calculates revenue, risks and labour input. Else, they could just look through
	the ways of achieving different money values w/o making any input.

	At this point there are planned: reading and writing into files
	(including creation of new objects there), use of Inheritance,
	design patterns, dynamic data structures, a kind of menu/UI
	considering current skill level.
